GUIDA PER L'USO DELL'APP CSV TO GOOGLE SHEETS

COME AVVIARE L'APP:
1. Estrai tutti i file dalla cartella ZIP (se hai ricevuto un file ZIP)
2. Per utenti Mac:
   - Fai doppio clic sul file "avvia_app.command"
   - Se appare un messaggio di sicurezza, vai su Preferenze di Sistema > Sicurezza e Privacy, e clicca su "Apri comunque"
   - IMPORTANTE: Usa sempre "avvia_app.command" e non l'eseguibile diretto per avviare l'applicazione
3. Per utenti Windows:
   - Fai doppio clic sul file eseguibile "CSV_to_GoogleSheets.exe"

COME USARE L'APP:
1. Una volta avviata l'app, ti verrà richiesto di autorizzare l'accesso al tuo account Google
   - Si aprirà una finestra del browser
   - Segui le istruzioni per concedere l'accesso
   - Copia il codice di autorizzazione e incollalo nell'app quando richiesto
2. Dopo l'autorizzazione:
   - Seleziona il file CSV che desideri convertire usando il pulsante "Seleziona file CSV"
   - Conferma la conversione facendo clic su "Converti in Google Sheets"
3. Una volta completata la conversione, riceverai un messaggio di conferma con il link al foglio Google creato

RISOLUZIONE PROBLEMI:
1. Se l'app non si avvia:
   - Mac: Assicurati di aver consentito l'esecuzione nelle impostazioni di sicurezza
   - Mac: Assicurati di usare "avvia_app.command" e non l'eseguibile diretto
   - Windows: Prova a eseguire come amministratore
2. Se hai problemi con l'autorizzazione Google:
   - Assicurati di avere una connessione internet attiva
   - Prova a cancellare i file token.json e token.pickle (se presenti) e riavvia l'app
3. Se il file CSV non viene convertito correttamente:
   - Verifica che il formato del file CSV sia valido
   - Assicurati che il file non sia troppo grande (limite di 5MB)
